## Test Case 1: Book a Demo with all required fields filled.
Description: Validate user being able to successfully submit a demo booking request by completing all required fields.
Preconditions: User has internet access, browser is supported and can open the brightsg website. User has valid email and phone number.
Test steps:
1.Navigate to website
2 .Click ‘book a demo’. 
3. Fill each box with required fields - first name, last name, email, phone, company name, selected product (BrightManager for this case)
4. Click book a demo button at the bottom of the form. 
5. Validate the demo has been booked with confirmation.
Expected results: All fields accept correctly formatted inputs. Book a demo button is clicked and success message appears: “Thanks for requesting a demo!”.
